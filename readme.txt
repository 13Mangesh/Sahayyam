step 1->install Angular 8  

step 2 ->install nodejs latest version

go to NodeJs folder in cmd and run following commands
0)npm init                   	(to generate package.json)
1)npm install body-parser    	(to parse data in node js)
2)npm install cors           	(for localhost connection from angular to node)
3)npm install express        	(for easy routes)
4)npm install -g nodemon     	(to start server automatically)
5)npm install dotenv 
6)npm install ejs 
7)npm install mongoose       	(for database connection)
8)npm install nexmo
9)npm install -g nodemailer-cli (for email notification)
10)npm install smpp

check weather all packages installed in  package.json  file inside depdendency section

1)inside mongodb create database with name Sahayyam
2)create collection donors
3)create collecton receivers

step 3->To start server 
Go inside  NodeJs folder in cmd and type "npm start"

step 4 ->To start project
go inside final folder in cmd and type "ng serve"
